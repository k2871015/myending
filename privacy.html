개인정보처리방침

최종 시행일: 2026년 5월 21일

Aetherius(이하 '에테리우스' 혹은 '사이트')는 사용자의 개인정보를 소중히 생각하며, 개인정보보호법 등 관련 법령과 구글 애드센스(Google AdSense) 프로그램 정책을 철저히 준수합니다. 본 방침은 당사가 수집하는 정보, 사용 목적, 그리고 안전성에 대한 규정을 다룹니다.

1. 인공지능 API 연동 및 브라우저 로컬 저장소(localStorage)

본 사이트는 사용자가 직접 발급한 Google Gemini API 키를 기반으로 맞춤형 시뮬레이션을 구현할 수 있습니다.

- 수집 및 보관 방식: 입력된 API 키는 당사의 중앙 서버로 어떠한 방식으로도 업로드되거나 수집되지 않습니다. 오직 사용자의 브라우저 내 안전 구역인 로컬 저장소(localStorage)에만 기록됩니다.
- 암호화 및 통신: API 호출 시 본 사이트의 프론트엔드 코드(JavaScript)가 구글 API 서버(generativelanguage.googleapis.com)와 직접 HTTPS 보안 암호화 세션으로 연결되어 데이터를 주고받습니다. 사용자는 원할 때 언제든 브라우저 설정이나 사이트의 API 설정창을 통해 이 키를 삭제할 수 있습니다.

2. 시뮬레이션 개인 정보 (이름 및 전화번호)의 안전성

본 시뮬레이터 이용을 위해 사용자가 입력하는 소유자 이름 및 전화번호는 오직 맞춤형 이야기 구성(개인화 시뮬레이션)을 위해서만 실시간 임시 세션 메모리 변수로 사용됩니다.

- 임시 보관 및 비저장성: 입력된 이름 및 전화번호 정보는 본 사이트의 데이터베이스나 어떠한 웹 서버에도 업로드되거나 저장되지 않으며, 사용자 로컬 저장소(localStorage)나 쿠키에도 영구히 기록되지 않고 오직 일시적인 브라우저 작동 세션 내에만 존재합니다.
- API 통신 시 안전성: 사용자가 본인의 Google Gemini API 키를 직접 등록하여 사용할 경우, 입력된 이름 및 전화번호는 오직 맞춤형 시나리오 생성을 위한 보안 SSL(HTTPS) 호출 시의 프롬프트 매개변수로만 구글 서버로 안전하게 전달되며, 제3자에게 유출되지 않습니다.

3. 광고 및 쿠키(Cookie) 수집 고지 (Google AdSense)

본 사이트는 신뢰할 수 있는 제3자 광고대행업체인 Google을 통해 맞춤형 광고를 제공받아 운영비를 충당합니다.

- 광고 서비스 제공: 구글을 포함한 제3자 판매자는 쿠키를 사용하여 사용자가 본 사이트 또는 다른 웹사이트를 이전 방문한 내역을 기반으로 광고를 게재합니다.
- 광고 쿠키의 사용: 구글은 광고 쿠키를 사용함으로써 구글 및 그 파트너가 사용자의 본 사이트 및 타 사이트 방문에 기초하여 관련성 높은 맞춤형 광고를 노출시킬 수 있게 합니다.
- 수집 거부 및 비활성화: 사용자는 구글 광고 설정 페이지를 방문하여 맞춤설정 광고 게재를 비활성화할 수 있습니다. (또는 aboutads.info를 방문하여 제3자 판매자의 맞춤설정 광고용 쿠키 사용을 중단시킬 수 있습니다.)

4. 자동 수집되는 정보 (로그 데이터)

당사는 사이트의 보안성 제고 및 통계적 트래픽 분석(Google Analytics 등)을 위해 방문자의 IP 주소, 브라우저 유형, 참조 링크, 페이지 체류 시간 등의 비식별 정보를 자동 수집할 수 있습니다. 이 정보는 순수한 사이트 분석용으로만 쓰이며, 어떠한 상황에서도 사용자 개인을 식별하거나 특정하는 데 사용되지 않습니다.

5. 개인정보의 보호책임자

개인정보와 관련된 의문 사항이나 쿠키 정책 변경 요구 사항이 있으실 경우, 본 사이트의 [문의하기] 페이지를 통해 즉각적으로 담당자에게 소통을 요청할 수 있으며, 당사는 최선을 다해 지원할 것입니다.
